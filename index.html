Welcome to the guide! We will be walking you through the process of installing an AWS server.

Begin by logging into you AWS account

Navigate to EC2

Select instances from the left panel

Select Launch instances

Scroll to Ubuntu Server 20.04

Select t2 (free tier in our case)

Select Review and Launch

Click Launch

Create new key pair, name the key someything memorable, click download. This will create a .pem file. Remember where you put it! We will use it later.

Click Launch, then View instances on the bottom right

click the checkbox next to your new server. In the window below under Public IPv4 address click the copy image. You can keep this page open just in case you loose your IP that is copied. We will need this later on in this guide

Success! You have gotten your server up and running! You can access it by inputting the IP address into your browser. Next you need to connect to it. For this we will use Putty.

install Putty. You want the link at the top of the page.

open Puttygen. It should have been installed with the putty package.

click load

find your .pem file

click save private key

for simplicity name it the same as the .pem file

open Putty

on the left click the plus next to SSH, then click on Auth. Click browse and find the new .ppk file you created with puttygen.

Now click on session on the top left. under host name place your IP address for the server you created. Hopefully it is still copied! Click open, and okay the next dialog box.

Since we made the server using ubuntu, your user name is 'ubuntu' a list to user names based on operating system is provided below.

For Amazon Linux 2 or the Amazon Linux AMI, the user name is ec2-user.

For a CentOS AMI, the user name is centos or ec2-user.

For a Debian AMI, the user name is admin.

For a Fedora AMI, the user name is fedora or ec2-user.

For a RHEL AMI, the user name is ec2-user or root.

For a SUSE AMI, the user name is ec2-user or root.

For an Ubuntu AMI, the user name is ubuntu.

For an Oracle AMI, the user name is ec2-user.

For a Bitnami AMI, the user name is bitnami.

Next we will install apache2

start with 'sudo apt update'

next, 'sudo apt install apache2'

restart and check the status of your apache2 server

'sudo systemctl restart apache2'

'sudo systelctl status apache2'

next we will navigate to your webroot folder

type 'cd /var/www/html'

now you can clone your github project to this destination and see it in real time.

type 'sudo git clone 'github repo url''

if you navigate to your ip address that was assigned earlier and add / the name of your projects folder you will be taken to the page.

success!
